# YouBikeProject
背景執行排程，定時取得政府公開的Youbike各站點即時資訊，提供歷史紀錄查詢頁面

- [x] 採用`前後端分離`的專案架構
- [x] 後端使用 .NET Core 3.1 開發
- [x] 前端使用 Vue 2.x + Ant Design Vue
- [x] 使用 Hangfire 處理背景排程工作
- [x] 使用 NUnit 測試框架
- [x] 採用 Dapper ORM 框架做數據庫操作
- [x] 數據庫使用 Heroku 的 PostgreSQL Cloud Databases
